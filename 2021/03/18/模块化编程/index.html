模块化编程

发表于 2021-03-18

（本博客模块化编程针对AT89C52单片机开发）
在比较大的项目中。程序的编写往往不是由一个人来完成。
为了提高效率，会把程序分成多个模块交给不同人来编写，最后再汇总起来。

在单片机的练习中，随着模块越来越多程序包含的函数也会越来越多，程序也会越来越长越来越复杂。这时通过模块化编程的方法可以大大简化程序。

前置知识

有些知识我认为对于学习模块化编程十分重要十分重要。通过学习这些知识可以更深入的理解模块化编程过程，从而更好地掌握模块化编程。

C语言源文件.c

我们编写程序时编辑的文件就是源文件。

C语言头文件.h

我们在C语言编写过程中必须写的#include中的stdio.h实际上包含了许多函数的声明，通过包含头文件的宏调用之后就可以使用头文件中的函数。

宏

我们常说宏定义，但真的理解宏吗？

宏定义

比如常见的

#define PI 3.14

用#开头的语句为预处理语句，#define语句的作用只是简单的字符替换。当程序开始编译时会进行宏展开，把#define语句作用域中的程序进行字符替换。

这里有几个注意的点：

1、#define的作用域是语句位置到程序末尾。主程序中可以定义宏。

int main(){
	#define A 12
    int c=A;
    printf("%d\n",c);
    return 0;
}

结果为

12

2、语句位置靠后的#define优先级高。

int main(){
    #define A 12
    #define A 10
    int c=A;
    printf("%d\n",c);
    return 0;
}

结果为

10

3、用#undef XXX可以取消XXX的宏定义

int main(){
    #define A 12
    #define A 10
    #undef A
    int c=A;
    printf("%d\n",c);
    return 0;
}

没有结果会在定义c的那行报错。

4、#define可以带参，可以用来定义简单的函数。

int main(){
    #define max(a,b) (a>b?a:b)
    printf("%d\n",max(1,4));
    return 0;
}

输出结果为4。

包含头文件的宏

就是头文件引用#include，也可以写成#include”stdio.h”。

两者区别如下：

“”：现在源代码文件夹中查找有无此头文件，如果没有则去编译器的bin文件夹查找。

<>：直接去编译器的bin文件夹中查找。

条件编译宏

我们一般用条件编译宏来包含头文件

比较常用的有格式是

#ifndef _SEG_H_ //如果没有的定义seg.h
#define _SEG_H_

#代码块

#endif
预处理宏

即已经被设置无法修改的宏。

比较常用的有DATE，TIME，FILE。

int main(){
    printf("%s\n%s\n%s\n",__DATE__,__TIME__,__FILE__);
    return 0;
}

结果如下

Mar 17 2021
16:32:28
C:\Users\86151\Desktop\C\hong\main.c

多文件编译原理

这是模块化编程的核心原理。

在一个.c文件转化为.xex文件的过程中。包含着编译（Compile）和链接（Link）两个步骤：编译负责检查源文件是否有语法错误、定义错误等问题并生成成目标文件（Object File）即Windows下的.obj文件。在编译的过程中对于源文件中的函数只会检查此函数是否被声明，并不会查找函数位置。链接则负责从多个.obj文件中查找函数位置。

所以模块化编程的过程如下：建立main.c文件和main.h文件以及各个模块的.c、.h文件比如seg.c，seg.h。.h文件中只有函数的声明，而.c文件中包含函数的具体定义。然后再main.c中包含main.h头文件，在main.h头文件中包含seg.h头文件。编译所有模块的.c文件最后再编译main.c生成.hex文件

Task

可以调节时间的clock，数码管显示。（仅当作模块化编程的例子）

main.c

#include "main.h"

int main(){
	uint8_t s=0,m=0,h=0;
	uint16_t cnt=0;
	while(1){
		deplay(s,m,h);
		cnt++;
		if(cnt==80)cnt=0,s++;
		if(s==60)s=0,m++;
		if(m>=60)m-=60,h++;
		if(h>=24)h-=24;
		value=0;
		get_key();
		if(value==1)h=(h+1)%24;
		if(value==2)h=(h-1+24)%24;
		if(value==3)m=(m+1)%60;
		if(value==4)m=(m-1+60)%60;
	}
}

seg.c

#include "seg.h"

//定义字符码表和位选码
uint8_t code segment_code[] = {
	0XC0,0XF9,0XA4,0XB0,0X99,0X92,0X82,0XF8,0X80,0X90 //对应0-9的字符
};//段码

uint8_t code bit_code[] = {
     0XFE,0XFD,0XFB,0XF7,0XEF,0XDF,0XBF,0X7F //对应8位数码管
};//位码

void deplay_2bit(uint8_t p,uint8_t n){
	
	BIT=0xff;
	SEGMENT=segment_code[n%10];
	BIT=bit_code[p];
	delay_ms(2);
	BIT=0xff;
	SEGMENT=segment_code[n/10];
	BIT=bit_code[p+1];
	delay_ms(2);
}


void deplay(uint8_t s,uint8_t m,uint8_t h){
	deplay_2bit(0,s);
	
	BIT=0xff;
	SEGMENT=0xBF;
	BIT=bit_code[2];
	delay_ms(2);
	
	deplay_2bit(3,m);
	
	BIT=0xff;
	SEGMENT=0xBF;
	BIT=bit_code[5];
	delay_ms(2);
	
	deplay_2bit(6,h);
}

key.c

#include "key.h"

sbit K1=P2^3;
sbit K2=P2^2;
sbit K3=P2^1;
sbit K4=P2^0;

//获得独立键盘的值
void get_key(){
	if(K1==0){
		delay_ms(5);
		if(K1==0)value=1;
		while(K1==0);
	}
	if(K2==0){
		delay_ms(5);
		if(K2==0)value=2;
		while(K2==0);
	}
	if(K3==0){
		delay_ms(5);
		if(K3==0)value=3;
		while(K3==0);
	}
	if(K4==0){
		delay_ms(5);
		if(K4==0)value=4;
		while(K4==0);
	}
}

//头文件

//粗略延时函数
void delay_ms(uint16_t ms){
    uint16_t i;
    uint8_t j;
    for(i=1;i<=ms;i++){
        for(j=1;j<=115;j++);
    }
}

注意

1、.h文件中不可以定义变量 ，在.h中只能声明，不能定义！

2、函数的声明有无extern都行，变量的声明必须加上extern，否则编译器无法识别声明

3、中断函数在.h文件中不需要声明
